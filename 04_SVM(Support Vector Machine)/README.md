## 🧠 Algorithms Covered

### 1️⃣ SVM Classification (SVC)
Used for classification problems.  
Finds the optimal hyperplane that maximizes the margin between classes.

### 2️⃣ SVM Regression (SVR)
Used for predicting continuous values.  
Fits a function within a specified error margin (epsilon) while controlling model complexity.

---

## ⚙️ Key Concepts

- Maximum Margin Hyperplane  
- Support Vectors  
- Kernel Trick (Linear, Polynomial, RBF)  
- Regularization Parameter (C)  
- Gamma (Kernel coefficient)  
- Epsilon-insensitive loss (for regression)
