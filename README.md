# Machine Learning Algorithms Implementation

This repository contains implementations of fundamental Machine Learning algorithms using Python and Scikit-Learn.  
Each algorithm is implemented in Jupyter Notebooks with additional visualization and Streamlit applications to help understand how the algorithms work.

The goal of this repository is to practice and demonstrate core machine learning algorithms through practical examples.

---

# Repository Structure

## 01 Linear Regression

Implementation of different variants of Linear Regression along with regression evaluation metrics.

**Contents**
- Simple Linear Regression
- Multiple Linear Regression
- Ridge Regression
- Lasso Regression
- Regression Metrics (MAE, MSE, RMSE, R²)

**Datasets**
- Bangalore House Price Dataset
- Placement Dataset

---

## 02 Logistic Regression

Implementation of Logistic Regression for classification problems.

**Concepts Covered**
- Binary Classification
- Sigmoid Function
- Probability Prediction
- Model Evaluation

---

## 03 K-Nearest Neighbors (KNN)

Implementation of the KNN algorithm for both classification and regression.

**Contents**
- Breast Cancer Classification using KNN
- House Price Prediction using KNN Regression
- Streamlit visualization app for KNN

**Key Concepts**
- Distance-based learning
- Choosing optimal K value
- Feature scaling

---

## 04 Support Vector Machine (SVM)

Implementation of Support Vector Machine for classification and regression.

**Contents**
- SVM Classification
- Support Vector Regression (SVR)
- Streamlit visualization for SVM decision boundaries

**Key Concepts**
- Maximum margin hyperplane
- Kernel trick
- Support vectors
- Regularization parameter (C)

---

## 05 Decision Tree

Implementation and visualization of Decision Tree algorithms.

**Contents**
- Decision Tree Classification
- Decision Tree Regression
- Overfitting vs Underfitting demonstration
- Tree visualization using dtreeviz
- Streamlit apps for classification and regression

**Key Concepts**
- Gini impurity
- Information gain
- Tree depth control
- Model interpretability
